#软件架构说明

 硬件平台：国信长天CT117E 

主控芯片：STM32F103RBT6

 操作系统：Windows 11 

开发环境：Keil-MDK 4.73 

软件开发包（SDK）：SPLib V3.5.0 

USB转串口驱动：FT2232驱动 

调试器下载插件：CoMDKPlugin-1.3.1.exe
